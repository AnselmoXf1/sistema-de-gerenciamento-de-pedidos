# 🚀 Início Rápido - Links Únicos

## Em 5 Minutos

### 1. Iniciar Sistema (1 min)

```bash
# Terminal 1
cd backend && npm run dev

# Terminal 2
cd frontend/cliente && npm run dev

# Terminal 3
cd frontend/painel && npm run dev
```

### 2. Gerar Primeiro Link (2 min)

1. Abra: http://localhost:5174
2. Login: `admin` / `admin123`
3. Clique: **"🔗 Gerar Link"**
4. Selecione: **"Delivery"**
5. Clique: **"Gerar Link Único"**
6. Clique: **"📋 Copiar"**

### 3. Testar Link (2 min)

1. Cole o link no navegador
2. Adicione produtos ao carrinho
3. Finalize pedido
4. Volte ao painel → Fila
5. ✅ Pedido deve estar lá!

## Atalhos Úteis

### Painel
- `Ctrl + 1` → Fila
- `Ctrl + 2` → Gerar Link
- `Ctrl + 3` → Histórico
- `Ctrl + 4` → Relatórios

### Gerador de Links
- `Tab` → Navegar campos
- `Enter` → Gerar link
- `Ctrl + C` → Copiar (após gerar)
