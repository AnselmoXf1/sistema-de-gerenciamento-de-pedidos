## 🚀 Como Rodar Agora

### 1. Instalação
```bash
# Backend
npm install

# Frontend Cliente
cd frontend/cliente && npm install && cd ../..

# Frontend Painel
cd frontend/painel && npm install && cd ../..
```

### 2. Configuração
```bash
# Copiar .env
cp .env.example .env

# Editar com teus dados
# DATABASE_URL, JWT_SECRET, etc
```

### 3. Base de Dados
```bash
# Criar tabelas
npm run db:migrate

# Popular dados
npm run db:seed
```

### 4. Rodar
```bash
# Terminal 1 - Backend
npm run dev

# Terminal 2 - Cliente
cd frontend/cliente && npm run dev

# Terminal 3 - Painel
cd frontend/painel && npm run dev
```

### 5. Acessar
- **API:** http://localhost:3000
- **Cliente:** http://localhost:5173
- **Painel:** http://localhost:5174
