# 🔗 Guia de Geração de Links Únicos

## Visão Geral

A funcionalidade de **Gerar Links Únicos** permite que a dona da lanchonete crie links personalizados para receber pedidos de diferentes formas:

- 🛵 **Delivery**: Para entregas
- 🥡 **Takeaway**: Para clientes levantarem no local
- 🪑 **Mesa Virtual**: Para pedidos online sem mesa física

## Como Funciona

### 1. Acesso à Funcionalidade

1. Faça login no **Painel Administrativo**
   ```
   http://localhost:517  ```

2. No menu superior, clique em **"🔗 Gerar Link"**

### 2. Configurar o Link

Na página de geração, você pode configurar:

#### **Tipo de Pedido**
- **Delivery**: Cliente receberá em casa
- **Takeaway**: Cliente levanta no balcão
- **Mesa Virtual**: Pedido online genérico

#### **Descrição (Opcional)**
- Adicione uma descrição personalizada
- Exemplos:
  - "Promoção do Dia"
  - "Cliente VIP"
  - "Evento Especial"
  - "Pedido Corporativo"

#### **Validade do Link**
- **1 hora**: Para promoções rápidas
- **6 horas**: Para período de almoço/jantar
- **24 horas**: Para o dia todo
- **7 dias**: Para semana toda
- **Sem validade**: Link permanente

### 3. Gerar o Link

1. Preencha as configurações desejadas
2. Clique em **"✨ Gerar Link Único"**
3. Aguarde alguns segundos

### 4. Compartilhar o Link

Após gerar, você terá várias opções:

#### **📋 Copiar Link**
- Copia o link para a área de transferência
- Cole em qualquer lugar (SMS, email, etc.)

#### **💬 WhatsApp**
- Compartilha direto no WhatsApp
- Mensagem pré-formatada com o link
- Ideal para enviar para clientes

#### **📤 Compartilhar**
- Usa o menu nativo de compartilhamento do celular
- Funciona com qualquer app instalado

#### **⬇️ Baixar QR Code**
- Baixa a imagem do QR code
- Use para imprimir ou enviar por email
- Formato PNG de alta qualidade

### 5. Cliente Faz o Pedido

1. Cliente recebe o link ou escaneia o QR code
2. Abre direto no cardápio
3. Seleciona os produtos
4. Finaliza o pedido
5. Pedido aparece na sua fila normalmente

## Casos de Uso

### 🛵 Delivery

**Quando usar:**
- Cliente liga pedindo delivery
- Promoção de delivery nas redes sociais
- Parceria com apps de entrega

**Como usar:**
1. Gere um link tipo "Delivery"
2. Envie para o cliente via WhatsApp
3. Cliente faz o pedido pelo link
4. Prepare e entregue

**Exemplo de mensagem:**
```
Olá! 👋

Faça seu pedido de delivery aqui:
[LINK]

Entregamos em até 45 minutos! 🛵
```

### 🥡 Takeaway

**Quando usar:**
- Cliente quer encomendar para levantar
- Evitar fila no balcão
- Pedidos antecipados

**Como usar:**
1. Gere um link tipo "Takeaway"
2. Compartilhe com o cliente
3. Cliente faz o pedido
4. Prepare o pedido
5. Cliente levanta quando estiver pronto

**Exemplo de mensagem:**
```
Olá! 👋

Faça seu pedido aqui e levante no balcão:
[LINK]

Sem fila, sem espera! 🥡
```

### 🪑 Mesa Virtual

**Quando usar:**
- Eventos especiais
- Pedidos corporativos
- Grupos grandes
- Catering

**Como usar:**
1. Gere um link tipo "Mesa Virtual"
2. Adicione descrição (ex: "Evento Empresa X")
3. Compartilhe com o grupo
4. Todos fazem pedidos pelo mesmo link
5. Prepare tudo junto

**Exemplo de mensagem:**
```
Olá equipe! 👋

Link para pedidos do evento:
[LINK]

Cada um faz seu pedido! 🍔
```

## Segurança

### Links Únicos
- Cada link tem ID único
- Não pode ser adivinhado
- Rastreável por origem

### Validade
- Links podem expirar
- Controle de tempo de uso
- Evita uso indevido

### Rastreamento
- Veja quantos pedidos por link
- Identifique links mais usados
- Analise efetividade

## Resolução de Problemas

### Link não funciona

**Problema**: Cliente diz que link não abre

**Soluções**:
1. Verifique se o link está completo
2. Teste o link você mesma
3. Verifique se não expirou
4. Gere um novo link

### QR Code não escaneia

**Problema**: QR code não é reconhecido

**Soluções**:
1. Verifique qualidade da impressão
2. Aumente o tamanho do QR code
3. Melhore a iluminação
4. Use fundo branco

### Pedido não aparece na fila

**Problema**: Cliente fez pedido mas não aparece

**Soluções**:
1. Verifique conexão com internet
2. Atualize a página da fila
3. Verifique se pedido foi finalizado
4. Consulte no histórico
